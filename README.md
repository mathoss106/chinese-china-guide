# Chinese China Guide

本项目是一个面向通用内容的 **独立 HTML 页面归档与发布仓库**。  
我们收集、整理并托管一系列自包含的 HTML 文件，每个页面均可独立访问，不依赖任何特定域名或单一网站。

## 项目简介

`chinese-china-guide` 旨在提供一个稳定、轻量的页面存储空间。  
仓库中的每个 HTML 文件均为完整页面，包含其所需的全部样式与脚本，便于长期保存与直接分发。

## 目录说明

```
.
├── pages/          # 归档的 HTML 页面文件
├── assets/         # 页面共用的资源（如图片、字体、样式等）
├── index.html      # 仓库首页 / 导航页
└── README.md       # 本文件
```

- `pages/`：存放所有独立 HTML 页面，按主题或时间组织。
- `assets/`：页面可能引用的外部资源，尽量保持精简。
- `index.html`：提供页面列表与快速跳转，便于浏览。

## 页面归档说明

- 每个 HTML 页面均为**自包含**的完整文档，可直接在浏览器中打开。
- 页面内容不针对特定网站或域名，适用于一般性的信息展示与资料存档。
- 归档的页面可能包含文本、图片、表格、交互元素等，但均遵循简洁、可读的原则。
- 我们不保证所有页面长期可用，但会尽力维护其稳定性与可访问性。

## 维护说明

- 本仓库由维护者不定期更新，新增或修正页面内容。
- 欢迎提交 Issue 或 Pull Request 以协助改进页面质量或添加新内容。
- 所有提交的页面应符合以下基本要求：
  - 有效的 HTML 结构
  - 无恶意脚本或外部依赖
  - 内容清晰、无侵权风险
- 仓库不提供任何形式的实时服务或 API，所有内容均为静态文件。
